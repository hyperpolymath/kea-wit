== Overview

Kea Wit provides interface definitions and tooling for WebAssembly Component Model development, following the Hyperpolymath language policy.

=== Intended Scope

* WIT interface definitions for Kea ecosystem components
* Tooling for WIT development, validation, and code generation
* Integration with the WebAssembly Component Model
